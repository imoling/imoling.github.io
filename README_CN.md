## ✦ 第一性原理 (The First Principles)

**iML Agent Desktop** 不仅仅是一个对话机器人，它是操作系统交互的进化形态。
专为那些不满足于简单对话，追求真正“掌控力”的用户设计——它跨越了 AI 推理与桌面执行之间的鸿沟。

> **“让每一台电脑都进化为拥有真实意志的桌面级智能体。”**

## ✦ 核心理念：Executive High-End

打破 AI 与操作系统的边界。iML Agent 拥有**原生钩子 (Native Hooks)** 以感知系统事件，并具备**全域执行 (Omni-Execution)** 能力来驱动工作流。

### 1. 全域感知 (Omni-Perception)
**它能看见你所见。**
监听文件变动、识别活跃应用上下文、读取系统通知及剪切板。它是智能体的**眼睛**。

### 2. 多维执行 (Multi-Execution)
**它能做你所需。**
从 Shell 脚本到 GUI 模拟，从浏览器自动化到原生应用 API 调用。它是智能体的**双手**。

### 3. 认知持久 (Vectra Persistence)
**它能记住一切。**
将系统中流转的所有碎片化信息，转化为可检索的本地向量知识库。它是智能体的**大脑**。

## ✦ 进化系统 (Evolution)

智能体与你共同成长。
- **技能生成**：一句话描述需求（如：“帮我压缩所有超过 1GB 的图片”），实时生成标准技能插件。
- **工作流编排**：通过自然语言交互，将技能编排为复杂的自动化工作流。

## ✦ 隐私第一 (Privacy First)

**Local First, Privacy Always.**
您的指令、数据、记忆与执行日志，100% 留存在您的本地。没有任何信息会被上传至第三方云端进行训练。
- 🛡️ 本地存储
- 🔒 端到端加密
- 👁️ 零追踪

## 🛠 技术栈

极致追求性能与美学。
- **核心**：Vanilla HTML5 / CSS3 (无框架依赖，纯粹性能)
- **动效**：GSAP (GreenSock Animation Platform) 打造“响应式动力学”
- **字体**：Inter & Plus Jakarta Sans & Outfit
- **设计**：深空黑 (#050505) 磨砂玻璃质感

## 🚀 快速开始

这是一个静态 Web/Landing Page 项目。
1. 克隆仓库。
2. 在浏览器中打开 `index.html`。
3. 或使用静态服务器预览：
   ```bash
   npx serve .
   ```
